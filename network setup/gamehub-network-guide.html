إعداد السيرفر (لابتوبك)

يجب تنفيذ هذا مرة واحدة فقط

1

شغّل الباك-إند على الشبكة كاملة

بدل ما يشتغل على localhost فقط، خليه يقبل اتصالات من كل الأجهزة:

cd C:\Users\%USERNAME%\Desktop\gamehub-backend-main\gamehub-backend-main
py manage.py runserver 0.0.0.0:8000
✅ هيك كل الأجهزة على الشبكة يقدروا يوصلوا للسيرفر
2

افتح البورت 8000 في الـ Firewall

شغّل CMD كـ Administrator:

netsh advfirewall firewall add rule name="GameHub" dir=in action=allow protocol=TCP localport=8000
3

اعرف IP لابتوبك

ipconfig

دوّن الـ IPv4 Address — مثلاً 192.168.1.100

⚠️ هذا الـ IP لازم تكتبه في setup.py على كل جهاز
4

حدّث ملفات الـ Startup على لابتوبك

عدّل start_backend.bat عشان يشتغل على الشبكة:

@echo off cd /d C:\Users\%USERNAME%\Desktop\gamehub-backend-main\gamehub-backend-main py manage.py runserver 0.0.0.0:8000
4

إعداد كل جهاز (×6)

كرّر هذي الخطوات على كل جهاز

1

ثبّت Python على الجهاز

https://www.python.org/downloads/ ☑️ تأكد من تفعيل: Add Python to PATH
2

انسخ مجلد files على الجهاز

من USB أو عبر الشبكة — حطه على Desktop

3

ثبّت المكتبات

py -m pip install requests pywin32
4

شغّل Setup Wizard

cd C:\Users\%USERNAME%\Desktop\files
py setup.py
📝 عند سؤال "عنوان السيرفر" اكتب IP لابتوبك مثلاً: http://192.168.1.100:8000
5

اختبر الاتصال

py gamehub_client.py
✅ يجب تشوف: تم فتح الجلسة — ID: X | الجهاز: PC-0X
6

أضف للـ Startup (تشغيل تلقائي)

افتح CMD كـ Administrator:

echo @echo off > "%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup\gamehub_start.bat" echo timeout /t 30 /nobreak >> "%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup\gamehub_start.bat" echo py C:\Users\%USERNAME%\Desktop\files\gamehub_client.py >> "%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup\gamehub_start.bat"
5

الاختبار النهائي

تحقق إن كل شي شغّال

1

شغّل الباك-إند على لابتوبك

py manage.py runserver 0.0.0.0:8000
2

شغّل الفرونت-إند على لابتوبك

npm run dev
📊 افتح Dashboard: http://localhost:5173
3

شغّل جهاز واحد وراقب الـ Dashboard

لما الجهاز يشتغل — رح تشوف الجلسة تفتح تلقائياً في الـ Dashboard بعد 30 ثانية

✅ إذا اشتغل — كرّر على باقي الأجهزة
4

أوقف جهاز وتحقق من إغلاق الجلسة

لما تطفئ الجهاز — الجلسة لازم تنغلق تلقائياً في الـ Dashboard

6

قائمة التحقق

اضغط على كل خطوة عند إتمامها

  • شراء راوتر 8 ports + 7 كابلات LAN
  • توصيل كل الأجهزة بالراوتر
  • معرفة IP لابتوبك (ipconfig)
  • فتح البورت 8000 في Firewall
  • تشغيل الباك-إند على 0.0.0.0:8000
  • تثبيت Python + files على جهاز 1 (PC-01)
  • تثبيت Python + files على جهاز 2 (PC-02)
  • تثبيت Python + files على جهاز 3 (PC-03)
  • تثبيت Python + files على جهاز 4 (PC-04)
  • تثبيت Python + files على جهاز 5 (PS-01)
  • تثبيت Python + files على جهاز 6 (PS-02)
  • اختبار فتح الجلسة تلقائياً على جهاز واحد
  • اختبار كل الأجهزة الـ 6 مع بعض
  • اختبار إغلاق الجلسة عند إطفاء الجهاز

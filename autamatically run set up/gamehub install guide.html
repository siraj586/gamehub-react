تثبيت Python

مطلوب للباك-إند والـ Auto-Client

1

حمّل Python من الموقع الرسمي

روح على الموقع وحمّل Python 3.10 أو أحدث

https://www.python.org/downloads/

☑️ Add Python to PATH — لازم يكون محدد!

تحقق من التثبيت

افتح CMD وشغّل:

py --version
النتيجة المتوقعة: Python 3.x.x

3

إعداد الباك-إند

Django API — gamehub-backend-main

1

افتح CMD وروح على مجلد الباك-إند

cd C:\Users\%USERNAME%\Desktop\gamehub-backend-main\gamehub-backend-main
2

ثبّت المكتبات

py -m pip install -r requirements.txt
3

هيّئ قاعدة البيانات

py manage.py migrate
4

أنشئ حساب المدير (Superuser)

py manage.py createsuperuser
سيسألك: Username / Email / Password
5

شغّل السيرفر

py manage.py runserver
✅ يجب تشوف: Starting development server at http://127.0.0.1:8000/
6

غيّر Role المستخدم إلى Owner

افتح المتصفح وروح على:

http://127.0.0.1:8000/admin

Users → اختر المستخدم → Role: Owner → ☑️ Staff status → Save

⚠️ بعد التغيير شغّل: py manage.py changepassword USERNAME
4

إعداد الفرونت-إند

React App — gamehub-react-main

1

افتح CMD جديد وروح على مجلد الفرونت-إند

cd C:\Users\%USERNAME%\Desktop\gamehub-react-main\gamehub-react-main
2

ثبّت الحزم

npm install
3

شغّل الفرونت-إند

npm run dev
✅ افتح المتصفح: http://localhost:5173
5

إعداد Python Auto-Client

يفتح الجلسة تلقائياً عند تشغيل الجهاز

1

ثبّت المكتبات المطلوبة

py -m pip install requests pywin32
2

روح على مجلد الـ files

cd C:\Users\%USERNAME%\Desktop\files
3

شغّل Setup Wizard

py setup.py
سيسألك: عنوان السيرفر / اسم المستخدم / اختيار الجهاز
4

اختبر السكريبت

py gamehub_client.py
✅ يجب تشوف: تم فتح الجلسة — ID: X | الجهاز: PC-01
5

أضف للـ Windows Startup (تلقائي)

افتح CMD كـ Administrator وشغّل:

أولاً — أنشئ ملف الباك-إند:
echo @echo off > "%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup\start_backend.bat" echo cd /d C:\Users\%USERNAME%\Desktop\gamehub-backend-main\gamehub-backend-main >> "%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup\start_backend.bat" echo py manage.py runserver >> "%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup\start_backend.bat"
ثانياً — أنشئ ملف الفرونت-إند:
echo @echo off > "%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup\start_frontend.bat" echo cd /d C:\Users\%USERNAME%\Desktop\gamehub-react-main\gamehub-react-main >> "%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup\start_frontend.bat" echo npm run dev >> "%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup\start_frontend.bat"
ثالثاً — أنشئ ملف الـ Client:
echo @echo off > "%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup\gamehub_start.bat" echo timeout /t 30 /nobreak >> "%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup\gamehub_start.bat" echo py C:\Users\%USERNAME%\Desktop\files\gamehub_client.py >> "%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup\gamehub_start.bat"
6

مرجع سريع — الأوامر اليومية

الأوامر اللي ممكن تحتاجها يومياً

▶️ تشغيل الباك-إند

py manage.py runserver

▶️ تشغيل الفرونت-إند

npm run dev

🔑 تغيير كلمة مرور

py manage.py changepassword USERNAME

👤 إنشاء مستخدم جديد

py manage.py createsuperuser

🗄️ تحديث قاعدة البيانات

py manage.py migrate

🎮 تشغيل الـ Client يدوياً

py gamehub_client.py

📋 مشاهدة الـ Log

type C:\Users\%USERNAME%\Desktop\files\gamehub_client.log

🌐 فتح الـ Dashboard

http://localhost:5173
7

قائمة التحقق

اضغط على كل خطوة عند إتمامها

  • تثبيت Python وتفعيل PATH
  • تثبيت Node.js
  • تثبيت مكتبات الباك-إند (pip install -r requirements.txt)
  • تشغيل migrate
  • إنشاء superuser وتغيير Role إلى Owner
  • تثبيت مكتبات الفرونت-إند (npm install)
  • اختبار الباك-إند على http://127.0.0.1:8000/admin
  • اختبار الفرونت-إند على http://localhost:5173
  • تثبيت مكتبات الـ Client (pip install requests pywin32)
  • تشغيل setup.py واختيار الجهاز
  • اختبار gamehub_client.py
  • إضافة ملفات الـ Startup
  • إعادة تشغيل الجهاز والتحقق من التشغيل التلقائي
8

مشاكل شائعة وحلولها

!

python was not found

Python مو مثبت أو PATH مو مفعّل

الحل: استخدم py بدل python
!

Invalid credentials

كلمة المرور غلط بعد تغيير الـ Role

py manage.py changepassword USERNAME
!

Error pausing/resuming session

الـ endpoint في AppContext.jsx غلط

الحل: غيّر /pause/ إلى /toggle_pause/ في AppContext.jsx
!

الجلسة ما تفتح تلقائياً

الـ Startup ما اشتغل أو السيرفر ما كان جاهز

الحل: تأكد إن ملفات الـ bat موجودة في مجلد Startup — شغّل: shell:startup
